## Installation

1. Clone this repository
   ```bash
   git clone https://github.com/gperakis/startup-metrics.git
   cd startup-metrics-dashboard
   ```

2. Install the required packages
   ```bash
   pip install -r requirements.txt
   ```

3. Run the dashboard
   ```bash
   streamlit run streamlit_app.py
   ```
